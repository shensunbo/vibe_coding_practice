# Game Development Skill

## Description
This skill provides assistance for 2D fighting game development using C++ and OpenGL.

## Capabilities
- Game loop implementation
- Character movement and animation
- Collision detection
- OpenGL rendering
- GLFW window management
- GLEW OpenGL extensions

## Usage
Use this skill to get help with game development tasks, such as implementing game mechanics, setting up rendering pipelines, and handling user input.
