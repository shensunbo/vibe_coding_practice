# 2D 格斗游戏依赖和工具清单

## 外部依赖

### 编程语言
- C++

### 图形库
- OpenGL

### 第三方库
- **GLFW**：用于窗口管理和输入处理
- **GLEW**：用于 OpenGL 函数加载

### 构建工具
- **Bazel**：项目构建系统

### 测试框架
- **Google Test**：单元测试框架

## 核心功能

1. **基本游戏框架**：窗口创建和基本渲染
2. **角色控制**：简单的移动和攻击
3. **碰撞检测**：基本的矩形碰撞检测
4. **主菜单**：简单的菜单界面
5. **战斗界面**：基本的战斗场景

## 项目结构
- 遵循 TDD 实践，测试代码与源代码分开组织
- 使用 Bazel 构建系统管理依赖和构建过程
- 代码注释使用英文，采用可生成文档的注释风格
